写一篇新笔记

用 Markdown 写正文，公式用 $...$ 或 $$...$$。3D 曲面可以拖进正文，也可以点击插入。

快速编辑工具栏

先在正文里选中一段文字，再点按钮，就能把它变成提示、定理、证明、双栏等区域；模块卡也可以直接点击或拖进正文。

灵感卡 适合记录突然想通的点
定理 醒目展示定义/定理
证明 给推导一个独立区域
问答卡 复习时很好用
配图 插入图片和说明
双栏 适合做公式/直觉对照

编辑已有笔记

从已发布列表中载入笔记。以后由本编辑器导出的页面会保留 Markdown 源数据，可以无损再编辑。
